Working Papers Database

The working papers database tracks recent working papers from the top 250 authors ranked by journal publications. This provides early insight into emerging research from leading scholars in finance.

Data includes: Author names, publication dates, venues (SSRN, arXiv, etc.), and citation counts for working papers.

About This Database

Data Sources

This database aggregates publication data from five leading finance journals:

  • JF - Journal of Finance
  • JFE - Journal of Financial Economics
  • RFS - Review of Financial Studies

Data is sourced from OpenAlex, an open catalog of scholarly papers, authors, and institutions.

Live Dashboard

For real-time updates and interactive data exploration, visit the Streamlit dashboard. The dashboard allows you to update data, filter by year, and explore working papers.
