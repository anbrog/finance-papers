About Finance Papers Database

Open source tool for tracking finance research

üìö What is this?

The Finance Papers Database is an open-source tool for tracking publications, citations, and working papers from leading finance and economics journals. It provides researchers, students, and practitioners with easy access to publication data and author rankings.

üìä Journals Covered

AER

American Economic Review - Top general economics journal

JF

Journal of Finance - Premier finance journal

JFE

Journal of Financial Economics - Leading finance research

QJE

Quarterly Journal of Economics - Top theory journal

RFS

Review of Financial Studies - Major finance publication

üîß How It Works

  1. Data Collection: Fetches publication data from OpenAlex API
  2. Storage: Saves data in SQLite databases for efficient querying
  3. Analysis: Ranks authors by publication count and citation metrics
  4. Working Papers: Tracks unpublished work from top authors
  5. Web Interface: Provides both dynamic (Streamlit) and static (GitHub Pages) access
